¿Que es C#?

C# (C Sharp) es un lenguaje de programación desarrollado por Microsoft, diseñado para ser simple, moderno y poderoso. Se ejecuta sobre el framework .NET y se usa para construir una gran variedad de aplicaciones

¿Por qué es importante?

  1. Alta demanda laboral
  2. Lenguaje moderno y versátil
  3. Lenguaje principal de Unity
  4. Buena integración con bases de datos, servicios web y la nube
