¿Que es Java?

Java es un lenguaje de programación de propósito general, orientado a objetos, multiplataforma y ampliamente utilizado en el desarrollo de:

  1. Aplicaciones de escritorio
  2. Aplicaciones web
  3. Aplicaciones móviles (especialmente Android)
  4. Sistemas empresariales (bancos, aerolíneas, etc.)

¿Por qué es importante?

Java se usa en millones de aplicaciones en empresas, gobiernos, sistemas bancarios, universidades, y mucho más. Aprender Java te abre la puerta a muchas oportunidades laborales.
