¿Que es Css?

CSS es un lenguaje de estilos que se usa para dar formato y diseño visual a páginas web escritas en HTML. Mientras que HTML estructura el contenido (por ejemplo, texto, imágenes, botones), CSS lo hace ver bonito: define colores, tamaños, posiciones, fuentes, márgenes, animaciones, etc.

¿Por qué es importante?

  1. Separa el contenido (HTML) del diseño (CSS).
  2. Facilita mantener y actualizar el diseño de un sitio web.
  3. Permite que una misma hoja de estilo afecte a muchas páginas (ahorrando tiempo y esfuerzo).
  4. Es esencial para hacer sitios web responsivos (que se adapten a móviles, tablets, etc.).
